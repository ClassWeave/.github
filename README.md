## 这个仓库放什么

按照 GitHub 的[组织 Profile 约定](https://docs.github.com/en/organizations/collaborating-with-groups-in-organizations/customizing-your-organizations-profile)，
组织里名为 `.github` 的仓库可以承载一组**面向整个组织生效的默认资产**。
ClassWeave 目前放了：

- `profile/README.md` — 组织主页正文。

未来按需要还可以补充：组织级 `CODE_OF_CONDUCT.md`、`SECURITY.md`、
`CONTRIBUTING.md`、`ISSUE_TEMPLATE/`、`FUNDING.yml` 等——欢迎以 PR 形式提议。
